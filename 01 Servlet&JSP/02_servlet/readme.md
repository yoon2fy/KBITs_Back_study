# JSP의 이해(심화1)
## Q1. index.jsp 파일에 대한 자동 변환된 서블릿을 찾아 내용을 확인하세요.
### A1. 
개발시 `index.jsp`가 `index_jsp.java`에서 `index_jsp.class`로 변환이 된다. jsp 파일은 내부적으로 java코드로 변환이 되고 class로 변환이 되기 때문이다. 즉, class가 실행 파일이므로 그렇게 변환이 되어야 하기 때문이다.
